明天，你好

发表于 2018-05-28 | 分类于 随笔

长大以后，我只能奔跑
我多害怕，黑暗中跌倒
《明天，你好》by牛奶咖啡

高中时代从班长的手机里第一次听到这首歌，很喜欢女声，很温柔，很温暖。

那时候没有手机，只能用一个很小很小的mp3在下了晚自习回家的路上不停的循环这首歌，边走路边哼哼，一遍一遍不厌其烦。

周末的时候无处可去，就会去学校上自习，那时候还是高一，教室里没什么人，就和徐君、叶君在黑板上写很丑的歌词，在教室大声唱跑调的歌。一边期待长大一边害怕长大。一边宣扬孤老一边暗恋姑娘。憧憬着以后，迷茫着以后。

那时候很迷恋学校书店里那种八块钱一本的青春散文，每天午饭以后都会去看看书店老板有没有进货，有就拿上一本，边回家边读，饶有滋味。去年暑假回家再翻出来读，却是已经变了味道，不禁笑起那年的年少多情。

与其一起翻出来的还有一本印刷的很细心的《文艺风象》，里面是些短篇小说，写的露骨。仍然记得那时候总是会在课间拿手肘杵一杵同桌，指着某段文字露出会心的笑容。其间意思大概就像古代的风流才子们结伴去青楼妓馆，某个经验丰富的风流才子向剩下的风流才子们推荐某个风姿绰约，才貌绝佳的勾栏美人时所说的话：“此女国色，弟不敢私”。粗俗一点就是“这娘们活贼他妈好，老弟我可不敢私吞”。

那一期的《文艺风象》里写了一个至今都难以忘怀的故事。故事名叫“羔羊之灯”，寓意取自《圣经》里的一句话“那城内又不用日月光照，因有神的荣耀作光，又有羔羊为城的灯”。写的颇有些悬乎。

大抵内容就是两个素昧平生的三流画家被雇来在一间很古老的寺庙里合作修复一副壁画。修复难度很大，要花的时间很长，两个人也都很穷，只能在寺庙里住下。画家A性格内向，很少说话，只在必要的时候和画家B交流上几句与壁画有关的事。画家B性格刚硬，却还是个善于顺应生活的主，对于画家A的爱搭不理并不放在心上。也正是因为这样的性格，他有个美丽能干的女人。每天他修复壁画回来，女人都已经准备好饭菜等他，生来命苦的男人白天干了一天，晚上到了床上还要接着干。日子一天天过去，男人在外工作，女人打理家务，偶尔女人也会问起住在隔壁的画家A，男人总是说那家伙是个闷油瓶，从来不说话。女人也就不再多问，心里却对画家A有了一点好奇，偶尔也会多做些菜给隔壁送去，画家A起初并不接受女人的好意，可是女人的那份执着以及娇滴滴的作态让他只得接受。女人的心里对他也越发的好奇，饭菜送的越发频繁了，每次见到他也都会笑着打招呼，因为她的善良所以他也会笑一笑回应。女人像是自己的魅力得到了证明一样开心，晚上在床上会故意提高叫声，以致她的男人会慌张的捂住她的嘴，隔壁还住着个男人呢…她自己却在心里暗暗的对调戏他感到喜悦和满足，白天的时候会进他的房间拿他的脏衣服出来洗，看见他乱糟糟的房间会帮他收拾好，觉得自己就像一个孩子的母亲。直到有一天她发现他的门上上了一把锁，她顿时羞红了脸，觉得自己受到了欺辱，她生气的敲门，他出来开门，她一把把他推倒在地上，走进他的房间哭了起来，他不知所措，支支吾吾也不知道说了些什么，她看见他这样哭的更凶了，开始指责了他起来，指责他不懂自己对他的好，一边说一边将身体逼迫过去仿佛要讨个说法。他更加慌张了，为了阻止女人靠近自己，拿起了身旁的剪刀企图吓退她，可是女人却没有退却 ，她继续靠近他的身旁，泪水中带着嘲笑，“你想要杀了我吗？你来呀！你来呀！”。两人靠的越来越近，直到紧紧相贴，男人惊慌的像个疯子，女人倒在了血泊中。

羔羊之灯，燃烧自己，照亮了谁呢？ ​​​​
